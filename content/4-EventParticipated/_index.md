## 1. Event: "Rational Learning (Dopamine) & Automated Prompt Engineering"

*   **Date:** 09:00, May 09, 2026
*   **Location:** Floor 26, Bitexco Financial Tower
*   **Role:** Attendee

### Event Description
This event was a unique intersection between **Neuropsychology** and **AI Technology**. The workshop focused not only on technical aspects but also delved into the most important "software": the human brain. Speakers shared methods to "hack" the Dopamine system to build self-discipline, combined with advanced techniques in controlling AI to optimize an engineer's performance.

### Key Takeaways & Accumulated Skills
*   **Brain Optimization (Cognitive Learning):** Understanding the "anticipation" mechanism of Dopamine, thereby applying the *Pomodoro* technique combined with *Digital Detox* to maintain a state of "Deep Work" – essential when researching complex software architectures.
*   **Systematic Prompt Engineering Mindset:** Shifting the approach to AI from asking general questions to systematically designing instructions following the **Role - Instruction - Context** standard.
*   **Advanced AI Technical Applications:** Exposure to techniques such as *Chain-of-Thought (CoT)* to solve logical security problems, *Tree-of-Thoughts (ToT)* to optimize code logic, and *RAG* to integrate system data into AI securely.
*   **Cost Optimization:** Gaining insight into *Token Economics*, learning how to structure Input/Output data to save resources when integrating APIs into SDLC projects.

*   **Date:** 09:00, June 06, 2026
*   **Location:** Floor 26, Bitexco Financial Tower
*   **Role:** Attendee

### Event Description
