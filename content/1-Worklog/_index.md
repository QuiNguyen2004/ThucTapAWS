On this page, I will present an overview of my Worklog throughout the **12-week** practical internship. 

The internship roadmap is divided into two main phases. The first phase (Week 1 - Week 7) focuses on researching, familiarizing, and practically configuring core Amazon Web Services (AWS) encompassing networking, computing, security, and cost management. The second phase (Week 8 - Week 12) involves applying all the acquired knowledge to architect and deploy a real-world project, **Pharmacare AI** – a medical E-commerce system integrated with an AI Chatbot (applying the RAG model).

Below is the detailed content of the tasks completed each week:

**Week 1:** [Cloud Computing Overview, Identity Management (IAM) & Storage (S3)](1.1-week1/)

**Week 2:** [Linux System Administration, Networking Basics & Command Line (CLI)](1.2-week2/)

**Week 3:** [Containerization Platform with Docker and Source Control (Git)](1.3-week3/)

**Week 4:** [Building Core Network Infrastructure (Amazon VPC) & Site-to-Site VPN](1.4-week4/)

**Week 5:** [Compute Administration (Amazon EC2) & Full-stack Web App Deployment](1.5-week5/)

**Week 6:** [High Availability Architecture Deployment (Auto Scaling & Load Balancer)](1.6-week6/)

**Week 7:** [Cloud Financial Management (FinOps) with AWS Budget & Cost Explorer](1.7-week7/)

**Week 8:** [AWS Serverless Architecture Design & AI Integration (Pharmacare AI)](1.8-week8/)

**Week 9:** [Database Design (ERD) & Vector Database Integration (pgvector)](1.9-week9/)

**Week 10:** [Building Serverless Backend API (AWS Lambda & API Gateway)](1.10-week10/)

**Week 11:** [Frontend Deployment, Authentication (Amazon Cognito) & Security (AWS WAF)](1.11-week11/)

**Week 12:** [CI/CD Pipeline Automation, System Testing, and Final Summary](1.12-week12/)
